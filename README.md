# polygot

## Installation guide
```sh
$ git clone https://github.com/sanketdhoble/polygot.git
$ cd polygot
$ virtualenv venv
$ source venv/bin/activate
$ pip3 install -r requirements.txt
```

## Run
``` sh
$ python manager.py
```
