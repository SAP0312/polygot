# polygot

## Installation guide
```sh
$ git clone https://github.com/sanketdhoble/polygot.git
$ cd polygot
$ virtualenv venv
$ source venv/bin/activate
$ pip3 install -r requirements.txt
```
=======

## Test
```sh
$ python manager.py test
```

## Run
``` sh
$ python manager.py
```

`localhost:8080/isTwoSidedPrime`
